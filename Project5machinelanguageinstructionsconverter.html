Appendix C: A Simple Machine Language

Op-code Operand Description
1 xy LOAD register R with the bit pattern found in the memory cell whose address is xy.
2 xy LOAD register R with the bit pattern xy.
3 xy STORE the bit pattern found in register R in the memory cell whose address is xy.
4 0 MOVE the bit pattern found in register R to register S.
5 xy ADD the bit patterns in registers S and T as though they were two's complement representations and leave the result in register R.
6 xy ADD the bit patterns in registers S and T as though they represented values in floating-point notation and leave the floating-point result in register R.
7 xy OR the bit patterns in registers S and T and place the result in register R.
8 xy AND the bit patterns in registers S and T and place the result in register R.
9 xy XOR the bit patterns in registers S and T and place the result in register R.
A xy ROTATE the bit pattern in register R one bit to the right x times. Each time place the bit that started at the low-order end at the high-order end.
B xy JUMP to the instruction located in the memory cell at address xy if the bit pattern in register R is equal to the bit pattern in register 0. Otherwise, continue with the normal sequence of execution.
C 00 HALT execution.
